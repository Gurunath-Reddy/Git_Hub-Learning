Git Commands (27-06-2024)

  1. git config --gloabl user.name/email :- To configure our details to GIT
  2. git init :-To initialize Git in the current directory
  3. git add :-To start track on the selected item
  4. git diff :-To Check differences after the changes
  5. git log :-To check Git logs
  6. git commit file_name -m "Commit message":-To commit the changes
  7. ssh-keygen -t rsa -b 4096 -C "your_email@example.com" :- To create SSH private and public key
  8. git Remote add origin https://github.com/yourusername/your-repository.git :- To build connection btw local Git and Remote Git-Hub
  9. git push -u origin main :- To connect with origin upstream and main branch and pushing the committed data from Local to Remote

Git-Hub (01/07/2024)

To Connect Git-Hub (Remote server - Host) with Git (local server - Tool) we need to follow this steps :-

  1. In Git-Hub we need to create a account.
  2. Create a repository and copy that path. (https://github.com/yourusername/your-repository.git)
  3. Now with using Git_bash we can connect our local(Git) with Remote(Git-Hub) with SSH Verification
  4. Generate SSH private and Public key using this command - ssh-keygen -t rsa -b 4096 -C "your_email@example.com".
  5. Now provide this Public key to Git-Hub and build the connection
  6. In VS code run this
    1. git Remote add origin https://github.com/yourusername/your-repository.git
    2. git push -u origin main
  7. To Push the content to Git-Hub , first we need to add that file and commit in local.
  8. After committing run this commad :- git push -u origin main (Here -u is used to specify stream/Remote(origin) and branch(main))
    After using git push "-u" origin main once, you can simply use: git push

Now all the data which are present Local repository are in sink with Remote Git-Hub
